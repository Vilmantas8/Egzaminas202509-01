### Git Workflow
- **main** - stabilus production kodas
- **feature/** - funkcionalumų vystymas
- Pull Request procesas visiems pakeitimams

### Commit konvencijos
- `feat:` - nauja funkcija

- `docs:` - dokumentacija
- `test:` - testai
- `refactor:` - refaktoringas
- `perf:` - performance gerinimas
